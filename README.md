 Job Market Analysis Using R

 Overview
This project analyzes a job market dataset using R Programming.

## Analysis Performed

- Top hiring companies
- Top hiring locations
- Industries with maximum job postings
- Top demanded skills
- Most frequent job titles
- Average salary by industry
- Highest paying job roles
- Experience vs salary analysis
- Salary vs applications relationship
- Monthly hiring trends

## Tools Used

- R
- RStudio
- dplyr
- ggplot2
- tidyr

## Project Files

- analysis.R
- datascience(3).csv
- top_companies.png
- top_locations.png
- top_skills.png
- salary_vs_applications.png
- monthly_hiring_trend.png

## Outcome

The project provides insights into hiring patterns, salary trends, skill demand, and industry growth.
